# Daily WebRTC Demo

React project demonstrating use of Daily.co Video Chat API.

Vite + React + Tailwind CSS starter based on [posva's](https://github.com/posva) [vite-tailwind-starter](https://github.com/posva/vite-tailwind-starter)

## Installation

```sh
yarn install
```

## Development

```sh
yarn dev
```
